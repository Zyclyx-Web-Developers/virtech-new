Next-Generation Firewall (NGFW)

A next-generation firewall (NGFW) is part of the third generation firewall technology that is implemented in either hardware or software and can detect and prevent sophisticated threats by implementing device, port and network security policies.

Unified Threat Management (UTM)

This is an information security term referring to a single security solution and system, offering multiple security functions at a single point on the network including features like antivirus, anti-spyware, anti-spam, network firewalling, detection and prevention of intrusion, filtering of content and prevention of leakage.

Intrusion Prevention Systems (IPS)

It is a network security framework that tracks malicious activity on the network or the system. Major intrusion prevention systems functions are to detect malicious activity, gather information about this activity, report it and try to block or stop it.

Web/Email Gateway Protection

Secure Email Gateways provide pre-delivery protection by blocking threats based on email prior to reaching a mail server. The gateway checks for signs of malicious or harmful material all incoming, outgoing and internal email messages, including attachments and URLs.

NAC/ AAA

Network access control systems enable Network monitoring and access management by implementing policies on devices and corporate network users. NAC solutions help organizations control access to their networks through Policy lifecycle management, Profiling and visibility, Guest networking access, Security posture check, Incidence response, Bidirectional integration. Authentication, Authorization and Accounting are the protocols that mediate network access.
